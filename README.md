# 📝 To-Do List em C#

Projeto simples de lista de tarefas feito em C# no console.

## 🚀 Funcionalidades
- Adicionar tarefa
- Listar tarefas
- Remover tarefa

## 🧠 O que aprendi
- Arrays
- Estruturas de repetição (for)
- Switch case
- Lógica de programação

## ▶️ Como executar
1. Abra o projeto no VS Code
2. Rode: dotnet run
